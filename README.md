# 2D Room Planner

A simple, browser-based 2D room planning tool built with vanilla HTML, CSS, and JavaScript.

## Description

This application allows users to design room layouts by placing and arranging furniture in a 2D space. It is a single-file application with no external dependencies.

## Features

- **Room Configuration**: Adjust room dimensions and grid unit size.
- **Furniture Management**:
  - Add rectangle or circle furniture items.
  - Customize dimensions and colors.
  - Resize and rotate items using handles.
  - Drag and drop to position items.
  - Delete selected items.
- **Grid Snapping**: Toggle snap-to-grid for precise alignment.
- **Data Persistence**: Export and import room layouts as JSON files.

## Usage

1.  Clone or download the repository.
2.  Open `index.html` in your web browser.
3.  Use the sidebar to configure the room and add furniture.
4.  Drag items to move them, and use the handles to resize or rotate.
